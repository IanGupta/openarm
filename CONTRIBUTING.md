# Contributing to OpenArm

Thanks for considering a contribution.

## Development setup
1. Install Node.js 20+
2. From repository root: `npm run setup`
3. Run app: `npm run dev:companion`

## Pull requests
- Keep PRs focused and small.
- Include screenshots/GIFs for UI changes.
- Update docs for behavior/config changes.
- Do not commit secrets, tokens, personal data, or machine-specific paths.

## Code style
- Prefer clear, defensive JavaScript.
- Keep platform-specific behavior behind explicit checks.
- Add comments where behavior is non-obvious.
