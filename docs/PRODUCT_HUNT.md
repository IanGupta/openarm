# Product Hunt Launch Kit — OpenArm

## Product name
OpenArm

## Tagline (<=60 chars)
OpenClaw companion for multi-device control

## Short description
OpenArm is a desktop companion for OpenClaw that adds Arm/Controller pairing, remote node operations, and simple multi-device setup workflows.

## First comment (maker comment)
Thanks for checking out OpenArm ??

I built this to make OpenClaw practical across multiple machines without complicated setup.

Core ideas:
- Controller + Arm role model
- Pair-PIN onboarding
- Remote node operations
- Release artifacts with SHA256 checksums

This is early and actively improving (especially cross-platform testing), so feedback and issues are very welcome.

## Q&A prep
### Who is this for?
People running OpenClaw across multiple machines who want quick pairing and practical remote workflows.

### How is it different?
OpenArm is agent-workflow-first (OpenClaw + nodes), not just a generic remote desktop app.

### Current limitations?
Cross-platform support is improving; Linux/macOS testing is lighter than Windows right now.
